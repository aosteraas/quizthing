# QuizThing

## Requirements

- Node.js >= 12
- PostgreSQL >= 10

## Setup

- Install Nest CLI `yarn add global @nestjs/cli` or `npm i -g @nestjs/cli`
- Install Lerna with `yarn add global lerna` or `npm i -g lerna`
- Clone repo
- Run `lerna bootstrap`
- Create your own `.env` in `packages/server` and point it to postgres
